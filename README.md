# Sri Usha Vet & Pet Stores

Production-ready full-stack e-commerce app for pet store business.

## Structure

- `client/` - Next.js frontend
- `server/` - Node.js backend
- `mobile/` - (optional PWA/app)

## Setup

See .env and .env.local for environment variables.

## Features
- Modern mobile-first UI
- Product catalog, cart, checkout
- User system, pet care, delivery
- Admin dashboard, reviews, offers
- Loyalty points, first-time discount, festival offers

## Deployment
- Frontend: Vercel/Hostinger
- Backend: Render/Railway
- DB: MongoDB Atlas
